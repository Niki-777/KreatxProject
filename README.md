Sistemi i Menaxhimit të Projekteve - Kreatx Internship
Ky projekt është pjesë e praktikës pranë Kreatx dhe shërben për menaxhimin e projekteve dhe taskeve.

Teknologjitë
Backend: .NET 10 & Entity Framework Core

Frontend: React JS

Database: SQL Server

API Documentation: Swagger

Arkitektura
Projekti është ndarë në tri shtresa kryesore:

KreatxProject.Server: Web API për logjikën e biznesit.

KreatxProject.Models: Class Library për modelet dhe validimet.

kreatxproject.client: Ndërfaqja e përdoruesit në React.

Si të niset projekti
Përditësohet Connection String te appsettings.json te projekti Server.

Ekzekutohet komanda Update-Database në Package Manager Console.

Pas nisjes së projektit, dokumentimi i API-t ndodhet në: /swagger/index.html.

Statusi i implementimit
Ndarja e suksesshme e modeleve në Class Library.

Konfigurimi i marrëdhënies One-to-Many mes Projekteve dhe Taskeve.

Implementimi i validimeve me Data Annotations.
